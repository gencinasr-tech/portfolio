## 🚀 Getting Started

### Prerequisites
- A modern web browser
- (Optional) Local web server for development

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/gencinasr-tech/portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio
   ```

3. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

4. **View the portfolio**
   - Open `http://localhost:8000` in your browser

## 📝 Customization

To customize this portfolio for your own use:

1. **Update Personal Information**
   - Edit `index.html` with your details
   - Replace profile picture in `assets/images/`
   - Update CV file

2. **Modify Styling**
   - Edit CSS variables in `assets/css/style.css`
   - Customize colors, fonts, and spacing

3. **Add/Remove Sections**
   - Modify HTML structure as needed
   - Update navigation links

4. **Update Translations**
   - Edit translation objects in `assets/js/script.js`

## 🔧 Configuration

### Color Customization
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00a8e8;
    --accent-color: #ff6b35;
    /* ... more variables */
}
```

### Language Settings
Default language can be changed in `script.js`:
```javascript
let currentLanguage = 'es'; // Change to 'en' for English
```
